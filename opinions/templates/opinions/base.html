{% load static %} {% block title %}DocketDrift{% endblock %} {% comment %} Analytics posture (2026-06-25): goatcounter ONLY -- no Google. The Author wants to know exactly TWO things, and nothing else: 1. What STATE is the visitor from -> which state to build next. 2. What DEVICE are they on -> how heavily to factor in mobile. That's the whole list. goatcounter derives location (state/region) and device/browser/OS server-side from the request; we deliberately collect NO behavior: - path is pinned to a CONSTANT ("/"), so goatcounter never learns which opinion/judge/search page a visitor viewed (no page-path tracking, even query-stripped -- which opinions you read is itself a research trail). - referrer is dropped (always ""). No Google, no search terms (search is POSTed -- query in the body, never a URL; access log is query-stripped too). Lawyers fear discovery of their research trail; we cannot produce what we never stored. Do NOT re-add path/referrer/ event tracking without the Author's say-so.
