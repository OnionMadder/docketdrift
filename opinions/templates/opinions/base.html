{% load static %} {% block title %}DocketDrift{% endblock %} {% comment %} Analytics posture (2026-06-25): goatcounter ONLY -- no Google. We send NO visitor data to Google (dropped GA4) and we NEVER record search terms. The only data points we collect are the ones used to decide which states to build next: country/region, language, device/browser, and the page path WITH ANY QUERY STRING STRIPPED. The path + referrer callbacks below hard-strip the query before goatcounter ever sees it, so even a stray ?q= that lands in a URL cannot be persisted. Search itself is POSTed (query in the request body, never in a URL) and the gunicorn access log is query-stripped too -- defense in depth. Lawyers fear discovery of their research trail; we cannot produce what we never stored. See CLAUDE.md "Data is sacred -- query privacy is non-negotiable" before touching anything analytics/logging-related. {% endcomment %} {% comment %} Canonical + Open Graph + Twitter cards.
